在ubuntu上部署mysql

发表于 2018-05-15 | 分类于 数据库

ubuntu版本:18.04
mysql版本:5.7.22
安装mysql
sudo apt-get install mysql-server
查看mysql版本
mysql -V
安装以后要修改root密码，打开/etc/mysql/debian.cnf
sudo cat /etc/mysql/debian.cnf
找到user和password，使用debian.cnf找到的user和password登录mysql，登录mysql以后
进入mysql数据库
use mysql
更新root密码
update user set authentication_string = password(‘yourpassword’) where user=’root’；
赋予外部机器访问本地数据库的权限
grant all privileges on . to root@”%” identified by “yourpassword”;
更新权限
flush privileges;
退出
exit
取消绑定本地地址
vim mysql.conf.d/mysqld.cnf
找到bind-address = 127.0.0.1，加上#注释掉
重启mysql服务
sudo service mysql restart

《架构到底是什么？》读后感

发表于 2018-05-02 | 更新于 2018-05-03 | 分类于 架构

以下是对文中观点的一些提炼：
架构设计与程序设计的区别：架构设计的思维是判断与取舍，程序设计的思维是逻辑与实现。
系统与子系统的区别：系统指的是一些互相关联的个体组成的群体，子系统是系统的一部分。
模块与组件的区别：模块是按逻辑划分，以职责分离为目的的单元；组件是按物理划分，以单元复用为目的的单元。
框架与架构的区别：框架关注的是“规范”，架构关注的是“结构”。
架构的定义：软件架构指软件系统的顶层结构。
总结：架构是顶层设计；框架是面向编程或配置的半成品；组件是从技术维度上的复用；模块是从业务维度上职责的划分；系统是相互协同可运行的实体

聊聊同步，异步，阻塞，非阻塞

发表于 2018-04-23 | 分类于 基础

同步与异步：这两个概念与通知机制有关，关注点是通知机制。同步指的是调用者需要等待被调用者的返回，是一种可靠的任务序列；异步指的是调用者无需等待被调用者的返回，是一种不可靠的任务序列，不能感知被调用者是否执行失败。

阻塞和非阻塞：关注点是在等待消息通知时的状态。这两个概念与程序（线程）等待消息通知(无所谓同步或者异步)时的状态有关。也就是说阻塞与非阻塞主要是程序（线程）等待消息通知时的状态角度来说的。阻塞就是在等待结果返回的时候，不能处理其他任务，处于等待的状态；非阻塞是在等待的时候，仍然能处理其他任务。
